# Personality Videos

Place your personality videos in this folder:

- `slave.mov` - Video to play when "slave" personality is selected
- `airdrop.mov` - Video to play when "airdrop_farmer" personality is selected
- `vet.mov` - Video to play when "vietnam_vet" personality is selected
- `gangster.mov` - Video to play when "gangster" personality is selected

## Supported Formats

- `.mov` (QuickTime) - Currently supported
- `.mp4` - Can be added if needed

The videos will automatically play fullscreen when the corresponding personality is selected, then return to chat when finished.

**Note**: Videos are muted by default to allow autoplay in modern browsers. The video will play automatically when the personality is selected.
